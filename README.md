# 朱一合 - 东方线香美学

一个优雅的线香销售网站，展现东方美学与意境。

## 项目特点

- 🎨 **东方美学设计**：淡雅配色，简约优雅
- 🌫️ **意境动画**：烟雾缭绕，增添意境
- 📱 **响应式布局**：完美适配各种设备
- 🛒 **购物车功能**：流畅的用户体验
- ✨ **交互动效**：精心设计的动画效果

## 技术栈

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- Vanilla JavaScript
- Google Fonts (Noto Serif SC)

## 页面结构

1. **首页**：品牌印象，烟雾动画
2. **品牌故事**：企业介绍，匠心传承
3. **产品展示**：6款线香产品
4. **香道文化**：焚香四境
5. **联系我们**：留言表单
